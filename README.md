#Digital-Humanities--Platt-F--Imperial-Cult-QGIS- 
Included in this repository is a link to a QGIS project. The CSV files in this repository are the files which were used to create the layers in the QGIS project. To use the project, these files must be uploaded to QGIS, as the layers are not saved when the project is downloaded. The CSV files can also be used as reference pieces to see the site names.
